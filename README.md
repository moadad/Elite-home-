# Elite Home Ware

نسخة متجر إلكتروني احترافية مبنية على نفس فكرة المشروع السابق لكن بهوية جديدة لمتجر **Elite Home Ware**.

## المزايا الأساسية
- بيع بالقطعة أو بالدستة أو بالاثنين معًا.
- الدفع الحالي: **كاش فقط**.
- الاستلام الحالي: **من أحد الفروع فقط**.
- لوحة التحكم تظهر بعد الضغط على اللوجو عدة مرات.
- متابعة الطلب برقم الطلب + رقم الهاتف.
- فاتورة جميلة مباشرة بعد إتمام الطلب.
- رفع صور المنتجات عبر Cloudinary.
- تثبيت المتجر كتطبيق PWA.
